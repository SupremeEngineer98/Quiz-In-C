# 🎮 C Quiz Game

A simple console-based multiple-choice quiz game written in C.  
This project was built to practice fundamental programming concepts such as arrays, loops, functions, and user input handling.

---

## 📌 Features

- Multiple-choice questions (A, B, C, D)
- Score tracking system
- Input validation for user answers
- Simple menu-based interface
- Lightweight console application

---

## 🧠 Concepts Practiced

- Arrays (1D / 2D char arrays)
- Loops (for / while / do-while)
- Functions
- Conditional statements
- Basic input validation
- Program flow control

---

## 🚀 How to Run

### 🔧 Compile the program

Using GCC:

```bash
gcc main.c -o quiz

```Windows
quiz.exe
